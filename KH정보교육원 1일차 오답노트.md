# 학습 오답노트(1)

#### 연습문제중 예시 코드를 보여주고 실행했을 때 런타임 에러가 일어날지 컴파일 에러가 발생할지에 대한 문제였습니다. 지금까지 java공부를 하면서 어느 상황에 컴파일 에러가 일어나는지 런타임 에러가 일어나는지 제대로 인식하지 못한 것 같아 박재할 겸 오답노트를 작성해 보겠습니다.
<br/><br/>

>컴파일 에러(compile Error)
#### -소스 작성시 문법을 잘못 작성해 프로그램을 컴파일 할 수 없는 에러를 말합니다.
#### -해결방법은 에러 메세지를 통해 에러가 발생한 부분을 확인할 수 있어 쉽게 해결이 가능합니다.
<br/><br/>

>런타임 에러(Runtime Error)
#### -프로그래머가 설계 미숙 즉 잘못된 코드 입력으로 프로그램 실행 중 발생하는 에러를 말합니다.
#### -해결방법은 해당 소스코드에서 발생하는 에러코드 부분으로 역추적해 원인을 확인해야 합니다.
<br/><br/>

>끝으로
* 컴파일 에러와 런타임 에러의 차이점을 알아보면서 간단한 문제인데 틀렸다는 것에 매우 창피했으며, 다시는 잊지 못할 것 같습니다.
