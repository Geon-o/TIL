#### 📝 2022 - 06 - 16 <script.js> UPDATE

- js파일과 css파일은 웹페이지상에서 제기능을 하지 못한다. 
- 즉 js, css파일들은 html파일을 통해서만 작동할 수 있다.
<br/>

- 변수 사용
    - 중복되는 데이터를 사용하거나 수정해야할 경우 값이 들어간 변수명을 사용하므로써 번거로움을 해결한다.

- Array
    - 배열의 변수명을 지정해주고 []안에 데이터를 넣어준다. 
    - 데이터가 들어가게되면 0부터 시작하여 데이터 개수만큼 자리값이 생성된다 자리 값을 index라고 한다.
    - 변수명.push를 사용하여 데이터를 넣을 수 있으며, index값을 이용하여 index자리에 있는 데이터를 꺼내올 수도 있다.

- Object
    - Array와 비슷하지만 Array는 하나의 정의만 가지고 있는 데이터들의 집합이라면,
    - Object를 이용하여 사용되는 집합은 각각의 property마다 다른 의미를 가지고 있다.
    - 즉 데이터의 값 설명이 필요하면 object를 사용해 정의해주면 된다.
        - object의 property를 통해 get,set이 가능하며, 이를 이용하여 property를 업데이트 할 수 있고 값을 추가 할 수 있다.
