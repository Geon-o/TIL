# Array

### 📍 
 - 데이터를 정리하여 취급하는 방법은 여러가지가 있다.
    1. List
    2. Array
    3. Record
    4. Map
 <br/>
 
 ### 1. List
  - 같은 종류(자료형)의 데이터를 차례 순서로 정렬해놓은 것이다.
  ```java
  var names = List.of("abc","def","ghi");
  ```
  - 위와 같이 배열에 넣어주고 .get(index)를 사용하여 특정 인덱스 값을 가져올 수 있다.
      - .get()메서드 말고도 다음과 같은 메서드도 사용가능하다.
          - 1. .size() => 배열의 크기
          - 2. .add() => 배열의 값을 추가(List는 배열의 크기가 정해져 있어 이와 같은 메서드를 사용하여 변경할 수 없다.)
 <br/>
 
 ### 1 - 1 ArrayList
  - List는 값이 정해져있어 수정하거나 추가할 수가 없다. 
  - 이러한 불편함을 ArrayList를 사용하면 해결할 수 있다.
  ```java
  var str = new ArrayList<String>()
  ```
  - 이와 같이 ArrayList객체를 new 연산자를 사용해 str변수로 지정해준다.
  - ArrayList를 사용하면 배열의 데이터를 추가, 변경, 삭제가 가능해진다. 즉 배열의 길이가 가변적이다.
  - 여기서 ArrayList를 사용할 때 다이아몬드 연산자("<>")를 통해 자료형을 지정해준다.
 <br/>
 
 ### 💡 제네릭
  - java에서 제네릭이란 데이터의 타입을 일반화 한다 라는 것을 의미한다.
  - 클래스나 메서드에서 사용할 내부 데이터 타입을 컴파일 시 미리 지정하는 방법이다.
       - 사용 방법은 위에 ArrayList예제에서 보았듯이 다이아몬드 연산자를 사용하여 데이터 타입(자료형)을 지정해 주었다. 
<br/>

### 💡 래퍼 클래스
 - 제네릭에서 다이아몬드 연산자에 기본자료형은 사용하지 못한다.
    - 제네릭은 기본형으로 취급하고 싶어도 사용하지 못하기 때문이다. 이에 기본자료형 값을 참조형으로 취급하기 위한 클래스가 래퍼 클래스이다.
        - 래퍼 클래스 종류
            1. Integer
            2. Double
            3. Boolean
            4. Character
        - 이와 같은 형태로 다이아몬드 연산자에 넣어 사용하면된다.
