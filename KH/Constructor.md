# 생성자(Constructor)

> Constructor
* 클래스로부터 객체가 생성됐을 때 자동으로 실행되는 특수 메서드로 멤버 변수 초기화에 사용된다.
<br/>

> 특징
* 작성 할때 클래스 명과 같은 이름으로 작성해야한다.
* 반환값이 없어 void를 사용하지 않는다.
* 필드 변수를 초기화 할 수 있다.
* 모든 클래스 내에 존재하며 작성하지 않아도 컴파일러가 자동으로 작성해준다.
<br/>

> 종류
1. 기본생성자
* 매개변수가 존재하지 않으며 구현부의 내용이 없다.
* 클래스 내부에 생성자가 없으면 컴파일러에 의해 자동으로 생성되는 생성자이다.
<br/>

2. 매개변수가 있는 생성자
* 매개변수가 존재하며
* 간결하고 직관적이다.
<br/>

##### 📍생성자에도 접근제한자를 사용할 수 있지만 그렇게 된다면 생성자를 사용하는 이유가 없어진다.
