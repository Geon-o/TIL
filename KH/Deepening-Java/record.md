# record

### 레코드란 데이터 클래스 이며 데이터를 취하기 위한 특수한 형태의 클래스이다.

### 💡
 - 보통 클래스 필드에 인스턴스 변수로 지정할때에 외부에서의 직접적인 접근을 막고자 private 접근제어자를 사용하게된다. 
 이러한 경우 사용자도 변수의 값을 변경할 수 없기에 매개변수가 있는 생성자를 통해 데이터 값을 받아 처리하게된다. 
 
 - 이떄 getter를 사용하여 데이터를 받게되는데 이러한 경우 코드가 매우 길어진다.
 <br/>
 
 ### 💡
 - 길어진 코드를 간결하게 표현하고 싶을때 record 클래스를 사용하면된다.
 ```java
 public record Example(String str, int num, Classtype class){구현부}
 ```
 - 이런 예제와 같이 매개변수에 필드명을 가져와 getter가 생성된다.
 - record 클래스의 구현부에 static 변수는 물론, static/public 메소드를 취할 수 있다.
<br/>

#### ✏ record클래스를 이용하여 훨씬 간결한 코드를 작성할 수 있으며 Entity 또는 DTO클래스 생성할 때 사용하면 유용할 것 같다.
