# Math Class

### 수학 관련 메소드 집합체이다. 보통 객체를 생성하지 않아 static메소드로 사용된다.

- 유일한 상수
    - Math 클래스에 유일한 상수 2가지가 있다.
        1. 자연로그의 밑을 나타내는 E
        2. 원주율을 나타내는 PI
<br/>

- 여러 메소드
    - 여태 사용했던 메소드 중에 round()메소드와 random()메소드도 Math클래스 안에 있다.
    - 이중 round()메소드는 반올림 기능을 하는데 반올림에는 rint()라는 메서드도 있다.
    <br/>
    
    - round() 와 rint()의 차이점
        1. round()메소드는 조건에만 충족하면 어느 값이던 무조건 올림을 해버린다.
        2. rint()메소드는 결과값의 가장 가까운 짝수를 반환한다.
    - round와 rint의 가장 큰 차이라면 round는 항상 올림하기 때문에 값의 오차가 크다. 이러한 오차범위를 줄이기 위해 rint를 사용한다.

<br/>

### ✏ 이외에 많은 메소드가 존재한다. Java API를 참고하면 좋겟다.
