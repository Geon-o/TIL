# Autoboxing & Unboxing

### jdk1.5 이전에는 기본형과 참조형간 연산이 불가능하였다. 그 이후에는 컴파일러가 알아서 Integer객체를 int타입으로 변환하여 연산이 가능해졌다.
<br/>

### 🏷
- Autoboxing
    - 기본형 값을 래퍼클래스의 객채로 자동 변환해주는 것을 말한다.

- Unboxing
    - 래퍼 클래스를 기본형 값으로 변환해주는 것을 말한다.
<br/>

### 💡
- Autoboxing과 Umboxing은 ArrayList클래스나 Vector클래스에 사용하면 값을 저장하거나 꺼낼 때 유용하게 사용될 것 같다.
