#set() Method , clear() Method

### 🏷 set()
- set() 메소드를 이용하여 날짜와 시간을 설정할 수 있다
- set() 메소드를 이용하여 날짜 전체를 설정할 수 있고, 년,월,일 따로 설정이 가능하다.
    - 주의❗
    - set()메소드에 월을 설정할 때 0 부터 시작한다. 0으로 부터 시작하는 이유는 배열을 사용하여 배열값을 가져오기 때문에 0부터 시작한다.
<br/>

- 시간을 지정할 때에는 시분초 전체를 설정할 수 없어 각각 지정해줘야한다.
<br/>

### 🏷 날짜와 날짜의 차이계산
- 계산해야 할 날짜와 날짜를 초단위(ms)로 변경하여 계산한다. 
- 결과값(ms)를 다시 시,분,초 or 년,월,일 로 변경이 가능하다.
- getTimeInMillis()메소드를 사용하여 초 단위로 변경해주고 1000으로 나눠줘야한다. 
- 일 단위로 변경하려면 하루(24*60*60)를 나눠 변경해준다.
<br/>

### 🏷 시간과 시간의 차이계산
- 시간도 초 단위로 변경하여 계산한다.
- abs()메소드로 절대값을 사용해서 계산한다.
- 결과값을 다시 시,분,초로 변경하려면 for문을 통해 변경해주고, 3600,60,1순으로 큰 값을 배열 앞에 두고 결과값이랑 나눠주고 나머지값은 그 다음 배열의 값이랑 나눠주며 계산해서 변경해준다.
<br/>

### 🏷 clear()
- Calendar객체의 모든 필드를 초기화시켜준다.
- 기존 getInstance()를 통해 현재시간을 설정해주지만 clear()메소드를 호출하면 1970년 1월 1일 00시 00분 00초(EPOCH Time)으로 초기화 시켜준다.
- clear(int field)의 Field값을 지정해주어 특정 필드만 초기화 할 수 있다.
