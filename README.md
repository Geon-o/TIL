### 📆2022 - 05 - 27
- 자바의 정석에서 배운내용들을 정리하려고 추가하였다. 이미 객체지향 부분은 학습했지만 계속 진도를 나갈 예정이다. 객체 지향부분은 정말 이해하기 어려운 부분인 것 같다. 꾸준히 반복학습을 통해 이해하고 익혀야 될 것같다. 그 부분도 따로 정리하여 기록할 예정이다. 
- 조만간 DB관련 학습도 하려한다. 생활코딩에서 MySQL관련 강의가 있다는 것을 알게되었다. 시간을 쪼개서 학습하고 요약정리하여 기록해보도록 할 것이다.

### 📆2022 - 05 - 28
- 국비교육 시작한지 2주 정도 지났다. 나름 강사님께서 열심히 설명하시는데 정리가 안되어 설명이 돼서 개인적으로 이해하기 어려운 부분이 많았다. 그래서 수업이 끝나면 따로 찾아서 정리하고 최대한 이해하려고 한다. 모자란 부분들은 책으로 보충할 것이다. 이번주까지 java 8버전에서 사용되는 것들을 학습하였는데 다음주부터 17버전에서 추가된 것들과 복습위주로 진행될 것 같다.
