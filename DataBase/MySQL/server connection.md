# MySQL server Connection

### 💡 MySQL에는 몇가지의 효용이 있다. 그 중 자체적인 보안체계를 가지고 있어 조금 더 안전하게 데이터를 보관할 수 있다. 또 권한기능이 존재한다.
<br/>

- MySQL에는 여러 사용자를 등록 할 수 있다. 
- 각각의 사용자에게 권한을 부여할 수 있다.
<br/>

### 📌서버 접속
- 명령 프롬프트 창을 열고 mysql -u (유저명) -p를 입력하면 패스워드를 입력하는 라인이 생긴다.
- 유저 생성시 패드워드를 생략하였음 그냥 enter입력 후 접속하면되고, 패스워드를 설정하였다면 패드워드 입력 후 접속가능하다.
    - 🚨 사용자 중 root 사용자가 있다. 이는 기본유저 즉 관리자를 말하는데 관리자는 모든 권한이 열려있어  root로 데이터베이스를 다루는 것은 위험하다.
