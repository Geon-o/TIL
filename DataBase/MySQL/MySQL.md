- file은 사용하기 쉽고 어디에나 존재하며 전송하기 편하다.
- 하지만 정보가 늘어남에 따라 file만으로 정보를 효과적으로 관리하는 것이 힘들어졌다.

### 관계형 데이터베이스
- 1970년 Edgar F. Codd라는 인물이 새로운 데이터배이스인 관계형 데이터베이스를 고안해낸다.
- 이러한 형태의 여러 관계형 데이터베이스 SW가 출시되기 시작한다.
    - 데이터를 표의 형태로 정리정돈이 가능해지며, 정렬, 검색같은 작업을 빠르고 편리하고 안전하게 사용할 수 있다.

<br/>

### MySQL
- 관계형 데이터베이스중 하나인 MYSQL은 1994년 스웨덴에서 개발되었다.
- 무료 오픈소스이면서 관계형 데이터베이스의 주요기능을 갖추고 있어 웹이 한창 성장할 시기에 개발자들에게 매우 좋은 대안이였다.
- 자연스레 수요가 많아지면서 웹과 함께 성장한 케이스이다.
